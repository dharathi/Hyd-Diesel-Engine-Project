## Kaggle dataset 

- Name: Dual-Fuel Engine Combustion Dataset
- Details: Contains 1463 records of data on performance and emission characteristics of a dual-fuel ICE operating under various load conditions and fuel ratios.
- URL: https://www.kaggle.com/datasets/ziya07/dual-fuel-engine-combustion-dataset
- Type of engine: Gasoline-Diesel hybrid engine
- Number of records: 1463
