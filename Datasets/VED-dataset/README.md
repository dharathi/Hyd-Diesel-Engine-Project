## VED Dataset

- Name: Vehicle Energy Dataset (VED)
- Details: Time-stamped naturalistic driving records of 383 vehicles (including gasoline ICE vehicles, EV, HEV, PHEV). Load changes and respective fuel consumption is recorded. The whole dataset has both static and dynamic data. For building the model only the dynamic data was considered. The static data has been uploaded in the same repository and the dynamic data can be downloaded from the Google Drive folder link below *Out of all the engines' data, only the hybrid engines' data (HEV & PHEV) was considered in this research.*
- Number of records: 21,960,500
- URL of source: https://github.com/gsoh/VED
- Research paper goals: The primary goal was to introduce and make publicly available the Vehicle Energy Dataset (VED), a massive collection of real-world vehicle operation data to fill a critical gap in research data for connected and autonomous vehicle (CAV) development.
- Google Drive URL to download the complete VED dataset: https://drive.google.com/drive/folders/1NhZOaxGk_r9cwq87bQLU4x-E94iXxrms?usp=sharing
